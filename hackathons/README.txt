DROP YOUR HACKATHON PHOTOS HERE
==============================

Save your photos in THIS folder with these EXACT names (all lowercase, .jpg):

  esith-week.jpg        -> the "WINNER" board photo (ESITH Week 2025) OR you at the ESITH lectern
  green-innovators.jpg  -> the Green Innovators' Showdown attestation
  iccitx.jpg            -> the IEEE ICCITX.0 finalist post screenshot

Rules:
- Name them EXACTLY as above (lowercase, hyphen, ".jpg"). A photo with any other name is ignored.
- Landscape / 16:9 looks best (they render as a banner on each card). Portrait works too, it just crops.
- Keep each file under ~1 MB so the site stays fast (resize if your phone photo is huge).
- If a file is missing, that card simply shows no photo — nothing breaks.

After adding them, rebuild + redeploy:  npm run build  &&  bash deploy.sh
